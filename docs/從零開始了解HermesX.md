# 從零開始使用HermesX

:::info
本篇文張將說明HermesX從開箱到上手的全部流程
同時也可以參考前頁所分享的說明影片
:::




### 電池安裝
- HermesX在剛拿到時，電池是跟主機分離的，你需要將18650塞入右側的電池倉中。
- 電池安裝完畢後裝置就會自己打開了
:::warning
- 尤其注意，在合上電池蓋前請務必確保**電池塞入到最底部**！
:::

### 關閉電源
HermesX上的旋鈕除了作為控制的輸入口以外，也是用來作為開關機的控制按扭！

- 在裝置運行時長按按鍵，側面的燈條會由上至下逐燈轉為紅色
- 隨著長按時間增加，當燈條全部轉紅後，螢幕會顯示“關機”字樣
- 然後就關機了


### 開機
**當你已經安裝好了電池，且裝置處於休眠狀態，此時長按裝置便可將裝置喚醒**
- 長按裝置，直到螢幕亮起，此時繼續長按
- 你會發現，隨著你長按的時間增加，螢幕上會畫出HermesX的LOGO
~~(很酷八，這段我花了6個小時寫)~~
- 等到你聽到開機音效，這時你才算正式將裝置開機完成！

:::info
實話說，其實長按只是控制裝置進入與退出休眠
不過休眠狀態下的耗電幾乎可以不計
裝置也不會對外發設任何無線電波
所以你們就把它當作關機就好ㄌ

:::

## LED及HermesX^UI^介面
### LED燈條的幾種回饋方式
| 狀態 | 顏色與動畫 | 說明 |
| --- | --- | --- |
| 待機 | 橘色燈條、有一顆亮點來回移動 | 裝置處於待命但可立即操作。 |
| 發送訊息 | 白色亮點自下而上流動 | 目前正在發送使用者的訊息。 |
| 接收訊息 | 白色亮點自上而下流動 | 收到其他節點的訊息。 |
| 收到節點資訊 | 綠色亮點自上而下流動 | 發現或更新網路節點資訊。 |
| 傳訊成功 | 綠燈閃爍三次 | 訊息已獲確認。 |
| 傳訊失敗 | 紅燈閃爍三次 | 訊息未成功送達，請重試。 |



### HermesUI使用說明

:::warning
這段我目前暫時是用AI寫的，基本上完全符合實際功能，但就是語氣跟口吻你們可能會覺得變的很“溫柔”。因為我真的沒力氣自己寫了，所以就麻煩見諒一下啦 

:::

HermesX 的操作很簡單，只要記住兩個動作就好：

- **旋轉旋鈕**：在不同頁面之間切換，或是在選單裡上下移動。
- **短按旋鈕**：進入目前選到的功能，或確認這一項。

可以把它理解成「先轉到你要的地方，再按下去」。

#### 一般選單怎麼用
平常開機後，你會先看到主畫面。這時候：

1. 短按旋鈕，你會進到“選單”
2. 旋轉旋鈕，可以在不同畫面之間切換。
3. 看到你要的頁面後，再短按旋鈕進入。
4. 進入選單後，繼續旋轉就是往上、往下選項目。
5. 再短按一次，就是確認或打開那個項目。

HermesX 最近的操作邏輯，已經把常用功能分成幾個固定入口，所以你實際使用時，會比較像「翻頁找到入口，再進去操作」，而不是在一大串設定裡面一直找。

#### FastSetup 怎麼用
FastSetup 可以把它想成「機器上的快速設定區」，適合你在戶外或臨時狀況下直接調整。

實際操作上：

1. 先旋轉旋鈕，切到 FastSetup 的入口頁。
2. **要進入 FastSetup，必須先旋轉**；如果只是短按，通常會直接跳到下一頁。
3. 進入後，你會看到一層一層的快速設定清單。
4. 旋轉旋鈕選到你要的項目，短按進去。
5. 調整完成後，再依照畫面提示返回上一層，或繼續調其他項目。

FastSetup 裡面會放比較常動到的東西，例如：

- UI 相關設定
- 節點常用設定
- GPS 相關設定
- 電源相關設定

如果你人在外面，不想每次都打開手機 App，優先先找 FastSetup 就對了。

#### GPS 怎麼看、怎麼調
HermesX 上的 GPS，不只是 App 裡能設定，機器本身也會有對應的畫面與快速設定入口。

從使用者角度，你可以這樣理解：

1. 先切到 GPS 頁面或進入 FastSetup 裡的 GPS 相關項目。
2. 先看目前 GPS 是不是已經定位成功。
3. 如果還沒定位，先把機器拿到空曠、看得到天空的地方，稍微等一下。
4. 有定位之後，再決定你要不要開位置廣播，以及多久送一次。

你如果需要調整，可以到`設定` > `裝置管理` > `GPS`：

- **GPS/定位開關**：決定裝置有沒有在使用 GPS。
- **廣播間隔**：多久把你的位置送到 Mesh 網路一次。
- **智慧位置**：如果你沒有明顯移動，就不要一直重複廣播，減少網路負擔。

這邊有一個很重要的觀念：

- 如果你把定位廣播設太頻繁，會增加整個網路的負擔。
- 如果你把位置精度設太高，也等於比較完整地把自己的位置送出去。

所以一般使用上，建議先以「夠用就好」為原則，不要一開始就把 GPS 廣播開得太積極。

#### 實機操作時的建議順序
如果你是第一次用 HermesUI，建議照這個順序摸一次：

1. 先在主畫面熟悉旋轉與短按的節奏。
2. 找到一般選單入口，試著進出幾次。
3. 再找到 FastSetup，看看裡面有哪些常用項目。
4. 最後再去看 GPS 畫面，確認目前有沒有定位、要不要開位置廣播。

這樣你之後在外面拿到機器時，比較不會手忙腳亂。


### 第一次連接你的行動裝置
Meshtatic 支援Android&IOS系統，在對應的應用商店都下載的到
詳細的使用教學可以參考Youtube影片，這邊就不贅述了。

1. 找到APP下方列表中的==裝置==
2. 此時你的手機會搜尋附近的Mesh裝置，在開機頁面中的右上角有你裝置的ID（例如==5d3c==) 

:::info
或是短按按鈕跳到裝置的倒數第二頁，右上角也會有
:::
4. 點擊對應的ID後，第一次連結都會需要輸入配對碼
5. 配對完畢後，APP會開始載入裝置裡的資訊，這時候不用管他
6. 等到裝置配對完畢後裝置頁面就會顯示配對成功了
7. 然後你就可以開始使用了！

### 頻道設定
在出貨前，每台HermesX裝置都已經設定頻道在台灣的MeshTW公用頻道裡了

==這邊建議除非有特殊需求，不然第一次使用的朋友直接使用即可==

如果需要設定自訂頻道的話（例如你要跟你的家人連線）
請看 [頻道設定教學](/-T0UZ81ETcC-eA3oOb2ppg)

:::warning
請注意！如果你將==位置==設為精確的話，將會將自己裝置的位置暴露在廣播當中
所以如果你不想這麼做的話，請設為模糊即可
:::

### 使用罐頭訊息
選轉選扭會跳出罐頭選單，壓下選扭則是送出對應的訊息
#### 自訂罐頭訊息
1. 到ＡＰＰ借面中的==設定==
2. 找到==罐頭模組==
3. 滑到最底部，找到==訊息==
4. 輸入你要設定的罐頭文字
:::info
每則罐頭文字都需要用 "|" 作為分隔（羅馬字符的分格號）
:::
